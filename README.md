## üìù Getting Started

### Step 1: Adding Transactions
1. Navigate to **Dashboard** and select **Record Transaction**
2. Enter the following information:
     - Description
     - Amount
     - Date
     - Type (Income or Expense)
     - Category
3. Click **Record Transaction** to save
4. Verify the entry appears in dashboard tables and home page metrics

### Step 2: Managing Investments
1. Navigate to **Dashboard** and select **Add Investment**
2. Input the following details:
     - Asset name
     - Investment amount
     - Expected return percentage
     - Investment date
3. Click **Add Investment** to confirm
4. Review projected returns in the investment portfolio table

### Step 3: Generating Financial Reports
1. Navigate to **Reports** section
2. Select the desired month to view detailed financial breakdown
3. Review the following information:
     - Income summary
     - Expense breakdown by category
     - Profit calculation
     - Salary distribution details
     - Recommendations and projections

### Step 4: Accessing Share Information
1. Navigate to **Shares** section
2. Review the shareholding pattern chart
3. Check share statistics including:
     - Current share price
     - Market capitalization
     - Book value per share
4. Analyze the 8-week share performance trend
5. Review detailed shareholder breakdown table

### Step 5: Contacting the Organization
1. Navigate to **Contact** section
2. Review organization details:
     - Physical address
     - Department-specific email addresses and phone numbers
     - Business hours
3. Complete the contact form with:
     - Your name and email
     - Organization name
     - Phone number (optional)
     - Subject category
     - Message content
4. Click **Send Message**
5. Confirm successful submission with the status message
